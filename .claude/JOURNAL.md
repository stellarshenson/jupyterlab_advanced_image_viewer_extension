4. **Task [Extended] - Fix image fit, zoom-out, nav; verify via Playwright** (v0.4.6): Reworked the viewer after the user reported broken fit (tiny/fragment/bottom-aligned images, worst on viewBox-only SVGs) and validated against acceptance criteria<br>

Per user, made nav arrows-only (hardcoded Left/Right, dropped `prevKey`/`nextKey` from `index.ts` + `schema/plugin.json`); nav opens next image and disposes the previous widget so one tab advances. Added the activation `console.log` back (Galata test) and fixed CI for frontend-only: `build.yml` drops the server-extension `grep` + `pytest` and adds badge `ignore_links`; `check-release.yml` gets `steps_to_skip: build-changelog` + `RH_SINCE_LAST_STABLE`.
